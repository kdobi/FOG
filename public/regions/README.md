# 대표지역 사진

실제 사진 파일을 이 폴더에 넣은 뒤 `src/main-v2.jsx`의 `REGION_PHOTOS`에 경로를 지정합니다.

예시:

```js
const REGION_PHOTOS = {
  jeonju: "/regions/jeonju.jpg",
  incheon: "/regions/incheon.jpg",
  daegwallyeong: "/regions/daegwallyeong.jpg",
  seoul: "/regions/seoul.jpg",
  chuncheon: "/regions/chuncheon.jpg",
};
```
